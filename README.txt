1. Deployment target is set to 7.0 
2. This app is tested and runs successfully on simulator(not tested on a actual device)
3. To obtain the proper results enter valid input dates
4. Valid dates:
- Format - mm/dd/yyyy
- mm must be between 1-12
- dd must be between 1-31 or 1-30 or 1-28 or 1-29(feb leap year) 
- yyyy must be from 1000- 9999
- All fields must be filled else a error text is displayed
- Non numeric values must be entered to compute date else corresponding error is displayed
5. The first date must be smaller  than second date 
6. the results are displayed on clicking calculate button
